# Local Data Persistence — UserDefaults & Core Data.

> **Draft:** This module is in progress. The lessons have been reviewed for familiarization purposes. The project is not fully completed.

An introduction to local data persistence on iOS, covering lightweight storage with UserDefaults and structured persistence with Core Data. This module explores how to save, retrieve and manage data, that persists between application launches.

## Project: Todo.

A task management application designed to practice data persistence techniques. The application allows users to create, read and manage a list of tasks with local storage.

### Key Concepts.

- `UserDefaults` for simple key-value storage.
- Core Data framework for structured data persistence.
- `NSManagedObject` and the Core Data stack.
- CRUD operations (Create, Read, Update, Delete).
- Data model design with `.xcdatamodeld`.
