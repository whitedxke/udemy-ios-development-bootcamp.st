# Xylophone.

A musical instrument iOS application, that plays sounds using AVFoundation. Each button triggers a different note, recreating a simple xylophone experience.

The main goal was to understand how audio playback works in Swift and to practice working with the `AVAudioPlayer` API. This project also focused on improving code structure and understanding the ViewController lifecycle.

----

## Key Concepts.

- `AVFoundation` framework and `AVAudioPlayer`.
- Functions, methods, and variable scope.
- Loops and data types.
- Error handling with `do..catch`.
- Debugging and refactoring techniques.
