# Egg Timer.

A simple iOS timer application, that helps boil eggs to the preferred level of doneness — soft, medium or hard.

The main goal was to learn, how to work with timers in Swift and apply different data types and control flow statements to implement the application logic.

This project served as a practical exercise in finding solutions independently using Apple Documentation, Stack Overflow and other resources.

## Key Concepts.

- Swift collection types, including `Dictionary`.
- `Timer` API for countdown functionality.
- Conditional statements: `if..else` and `switch`.
- Functions with return values.
- `ProgressView` for visual feedback.
